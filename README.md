# MasterControl MSBA Capstone Project

## Overview  
This project was completed as part of the MSBA capstone in collaboration with MasterControl, a B2B SaaS company focused on quality and manufacturing software solutions. The work applies data analytics to a real business problem to generate actionable insights and support decision-making.

## Business Problem  
The goal of this project was to analyze sales and lead data to better understand conversion behavior and improve lead prioritization. The key challenge was identifying how to more effectively allocate sales efforts toward leads with higher likelihood of conversion.

## Approach  
The analysis included exploratory data analysis, data cleaning, and predictive modeling. Key patterns in the data were identified through visualization and statistical exploration, and modeling techniques were used to estimate conversion likelihood and support prioritization strategies.

## Key Value  
The project provides insights into lead behavior and highlights opportunities to improve sales efficiency through data-driven decision-making. It demonstrates how analytics can be used to better prioritize opportunities and support business performance.

## Tools Used  
Python, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn, Jupyter Notebooks
